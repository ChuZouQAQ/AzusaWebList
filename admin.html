☁ GitHub 同步（用于在 Vercel 上持久化）

配置后可一键 commit & push assets/data.js 到 GitHub。Vercel 会自动重新部署。
Token 只保存在你本机浏览器的 localStorage，不会传给别人。

📖 如何获取 Token？
1. 打开 github.com/settings/personal-access-tokens/new（细粒度 Token，更安全）
2. Repository access → Only select repositories → 选你这个仓库
3. Permissions → Repository permissions → Contents: Read and write
4. 生成后把 Token 粘到上面，点「保存配置」
或者使用 classic token，权限选 repo 即可
